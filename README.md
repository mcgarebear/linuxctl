# linuxctl

`linuxctl` enables you to quickly bootstrap a Linux development environment
based on
[Ubuntu 24.04 LTS](https://ubuntu.com/download/desktop).
`linuxctl` manages the common packages, directory structure, and shell
environment.

[Raspberry Pi 5](https://www.raspberrypi.com/products/raspberry-pi-5/).

## Install

```bash
mkdir --parents "${HOME}/code/github.com/mcgarebear"
git clone git@github.com/mcgarebear/linuxctl.git "${HOME}/code/github.com/mcgarebear/linuxctl"
export PATH="${HOME}/code/github.com/mcgarebear/linuxctl:${PATH}"
```
